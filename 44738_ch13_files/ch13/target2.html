Chapter 2: Using Basic Selectors

Choosing the right selector is the key to working successfully with CSS. Although the CSS3 specification defines more than 40 selectors, not everyone uses the latest browser. This chapter concentrates on a smaller range of selectors supported by IE 7, which form a solid basis for styles recognized by the vast majority of browsers.

Part II: Working with Text

Chapter 4: Styling Text

Presenting text in a visually appealing manner is an important part of web design. Text must not only look good, but it must be easy to read. This chapter covers the full range of text properties—choosing fonts, font sizes, colors, weight, and spacing. It covers all the CSS2.1 text properties plus widely supported CSS3 properties, including adding shadows to text.

  • Selecting and adjusting fonts
  • Changing the color of text
  • Formatting text

Chapter 5: Embedding Web Fonts

For years, web designers have been frustrated by the narrow range of “web-safe” fonts. Although Internet Explorer 4 introduced the technology for embedding fonts as far back as 1997, licensing issues stood in the way of its widespread adoption. The licensing issues have now been resolved, making it possible to use a much wider range of fonts with the CSS @font-face rule.

Part III: CSS Page Layout

Chapter 6: Understanding the CSS Box Model

The CSS box model lies at the heart of current layout techniques. This chapter explains how browsers handle width, height, and vertical and horizontal spacing in both inline and block-level elements. It also shows how to use the CSS display property to change the default way elements are treated. Brief mention will also be made of the CSS3 box-sizing property, which changes how width and height are calculated for individual elements.

  • Introducing the CSS box model
  • Understanding the difference between padding and margin
  • Specifying minimum and maximum widths and heights
  • Using the display property to change how elements are displayed
  • Controlling overflow and visibility
  • Can you turn the clock back with box-sizing?

Chapter 7: Floating Elements for Layout

CSS3 promises to change the way web pages are laid out, but it will be many years before the proposed grid and flexible box layout modules are universally supported. In the meantime, floating elements to sit alongside each other horizontally remains one of the most important layout techniques. This chapter explains how floats work, and highlights how to deal with common problems.

  • Flowing text around images
  • Centering images
  • Tackling problems with floats
