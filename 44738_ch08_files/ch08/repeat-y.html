Vertical Tiling

In this page, background-repeat has been set to repeat-y, so the background image tiles down the side of the page.

The background image is 60px wide, so the body has been given a left margin of the same width to center the content in the remaining space. Note that the background extends into the left margin. Normally, backgrounds don't extend into the margins of an element. The exception is the <body> element, which acts as the canvas for the whole page.
