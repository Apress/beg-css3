The border shorthand property also takes a space separated list of color, style, and width values, which can be in any order. The same value is applied to all four sides of the element. If a value isn’t specified, the initial value for the individual property is applied.

If you want the same border on all four sides, the border shorthand is clearly the best option. For example, the following definition creates a 4px double red border around an element:

border: 4px double #F00;

It couldn’t really be much simpler.

Tip The default border style is none. If you forget to specify a style, no border is created, even if you set a color and/or width. To make a border invisible, but preserve its width, set its color to transparent. This can be useful to prevent an element from moving if you display a border only when you mouse over it.
